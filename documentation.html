API Endpoints

To get started, use the following endpoints:

  • GET /api/words - Retrieve a list of crypto terms.
  • GET /api/words/{term} - Get the definition of a specific crypto term.
  • POST /api/words - Add a new crypto term with the following payload:
{
    "term": "exampleTerm",
    "definition": "This is an example definition for the term."
}

Examples

Here are some example requests:

GET /api/words
GET /api/words/bitcoin

POST /api/words
Request Body:
{
    "term": "bitcoin",
    "definition": "A digital currency created in January 2009 following the housing market crash."
    "example": "btc"
}
